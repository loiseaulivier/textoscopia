# Textoscopia

**Textoscopia** is a simple, browser-based tool for corpus linguistics analysis. It requires no installation, no server, and no technical expertise — just open the HTML file in any modern browser and upload your documents.

## What it does

Textoscopia allows you to explore one or more PDF or Word (.docx) documents using a full set of corpus linguistics methods:

- **KWIC** (Keyword in Context) — see every occurrence of a word surrounded by its context, with a configurable window size
- **Word frequency** — ranked list of the most common words in the corpus
- **Collocations** — find which words habitually appear near a node word; click any collocate to see all concordance lines
- **Pattern search** — use regular expressions to find linguistic patterns, shown in KWIC-style context, with a built-in regex help panel
- **N-grams** — find the most frequent sequences of 2 to 5 consecutive words
- **Dispersion plot** — visualise where in the corpus a word appears
- **Type/Token ratio** — measure lexical diversity using TTR and MATTR
- **Keyword analysis** — with two or more files, identify statistically distinctive words using log-likelihood
- **Export** — download the full extracted text as a plain .txt file

All processing happens in the browser. No data is sent to any server.

## How to use it

Download `textoscopia.html` and open it in any modern browser (Chrome, Firefox, Safari, Edge). No installation required.
